Журнал учета работ КТСМ (АРМ Электромеханика)
Приложение на Windows Forms (C#) для автоматизации учета технического обслуживания устройств КТСМ (Комплекс технических средств мониторинга) на железнодорожных станциях. 
Программа выполняет роль электронного журнала, фиксируя время начала и окончания работ, исполнителей и тип оборудования, с возможностью выгрузки отчетов в Excel.
